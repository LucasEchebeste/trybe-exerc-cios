Fatos interessantes sobre o Stomatopoda

Seu nocaute equivale a um tiro de pistola calibre 22. Como se não bastasse, estes crustáceos apresentam a melhor visão em cores do mundo. Possuindo 08 fotorreceptores para distinção de cores (humanos possuem 03), conseguem detectar ondas ultravioleta e milhares de gradações (até 16 cores primárias).
